Projects Development About

Apple Developer Academy @ POSTECH  —  iOS Development

During the 9-month training at the Apple Developer Academy, I intentionally distanced myself from design to focus entirely on iOS development from a developer's perspective. During this period, I participated in four team projects, taking charge of iOS development for all of them, and released two apps on the App Store.

Wote - A Voting Community App Helping Teenagers with Consumption Worries

I was responsible for the front-end development of a community platform app that helps teenagers make rational consumption choices by sharing their worries with peers and establishing their consumption values by observing others' consumption tendencies. I gained experience in handling various data communications by collaborating with the backend team.
  • Implemented Apple login and managed user JWT (JSON Web Tokens) using Keychain.
  • Created an API Manager for server communication using Alamofire, Moya, and Combine, and managed cases separately.
  • Collaborated with the backend team to connect each API to the view and designed the architecture focusing on data.
  • Implemented a custom image crop feature (zoom in/out, move, rotate, crop).
  • Enhanced readability and reusability by separating views into computed properties, methods, and structures.
  • Defined and managed conventions using Github Issue, Pull Request, Review, and SwiftLint for collaboration.
  • Directly handled the process of submitting the project for App Store review and release.

Soook - A Game App Performing Various Punishment Missions

I was responsible for the front-end development of a game app where users shake a bubble tea to randomly select a person to perform various missions, making it easier for friends to get closer while having fun. I gained experience in implementing smooth and natural animations for various views.
  • Implemented animations for naturally inserting a straw according to user gestures and animating random pearl views.
  • Implemented handling of Korean particles ‘이' and ‘가' by separating Hangul syllables into initial, medial, and final sounds and using Unicode.
    •
  • Adopted MVVM architecture for the app and handled data binding for randomly selected mission data.
  • Implemented a read-along mission view using STT (Speech-To-Text) functionality.
  • Facilitated smooth collaboration among developers using Git and GitHub.

AllerView - An App for Checking Allergy-Inducing Ingredients for Foreigners

I was responsible for the front-end development of a utility app that helps foreigners visiting Korea register their allergies and check for allergy-inducing ingredients in food by taking pictures of the ingredient labels. I gained experience in handling API requests and responses using the ChatGPT API.
  • Connected to OpenAI's ChatGPT API using Swift Concurrency, Alamofire, and URLSession.
  • Conducted prompt engineering to parse ChatGPT's results into the desired format for use.
  • Recruited actual foreigners to conduct user interviews and app testing.
  • Managed the entire project using Notion to handle tasks and documents.

WallaRun - A Drug Prevention Educational Game App for Teenagers

I was responsible for the front-end development of an educational game app emphasizing the dangers and addictiveness of drugs to prevent drug use among teenagers. I gained experience in handling game graphics and physics engines using SpriteKit, a framework provided by Apple.
  • Implemented game views and GameScene using SwiftUI and SpriteKit.
  • Developed the running and jumping abilities of the wallaby character.
  • Implemented obstacle generation, collision detection, and reduction of life and abilities upon collision.
  • Created moving backgrounds and defined the finish line in the game.

Developer's Perspective

Studying iOS development and working on actual development projects provided a deep understanding of the entire development process and environment, and how design is implemented. I also gained experience in recognizing the differences in perspectives between developers and designers and learning how to communicate effectively. These experiences enabled me to approach problems from both development and design perspectives and derive optimal solutions.
