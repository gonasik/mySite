Apple Developer Academy @ POSTECH  —  iOS Development

Studied iOS development for 9 months at the Apple Developer Academy, focusing solely on development from a developer's perspective, intentionally distancing myself from design. During this period, I completed a total of four team projects, and in all projects, I was responsible for iOS development. I released a total of two apps on the App Store.

Wote - A Voting Community App Helping Teenagers with Consumption Worries

Developed the front-end of a community platform app that helps teenagers make rational consumption choices by sharing their worries with peers and establishing their consumption values by observing others' consumption tendencies. I collaborated with the backend team to handle various data communications.
  • Implemented Apple login and managed user JWT (JSON Web Tokens) using Keychain.
  • Created an API Manager for server communication using Alamofire, Moya, and Combine, managing cases separately.
  • Collaborated with the backend team to handle complex data communications, improving the user experience.

Relay - A Time Management App for Teams

Developed the front-end of an app designed to manage and share team schedules efficiently. This app allows team members to set deadlines, share tasks, and monitor progress in real-time.
  • Integrated calendar API for real-time schedule synchronization.
  • Used SwiftUI to create a user-friendly interface that dynamically updates with task status.
  • Implemented push notifications to remind team members of upcoming deadlines.

Explore - A Travel Planning and Sharing App

Led the front-end development of a travel app that allows users to plan trips, share itineraries with friends, and discover popular destinations based on user reviews.
  • Implemented map integration using MapKit to display travel destinations and routes.
  • Developed a feature to share travel plans with friends and receive feedback in real-time.
  • Used CoreData to manage and store travel data locally on the device.
